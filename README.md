# CryptoPulse

CryptoPulse is a desktop-focused cryptocurrency tracker built with Next.js.  
It shows live cryptocurrency market data in a clean and simple interface.

The goal of this project is to practice building a modern web application using
Next.js, external APIs, and component-based design.

---

## Features

- View live cryptocurrency prices
- See top cryptocurrencies by market capitalization
- Search coins by name or symbol
- View detailed information for each coin
- Dark mode by default with optional light mode
- Desktop-first layout
- Live data fetched from the CoinGecko API

---

## Technologies Used

- Next.js
- React
- Tailwind CSS
- CoinGecko Public API
- Vercel (deployment)

---

## Pages

- Home page with coin list and search
- Coin details page with price information
- About page
